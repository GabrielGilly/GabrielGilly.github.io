Documenting my journey

2021-05-06

Why starting a blog in 2021? It can seam out of place in the age of youtube and podcast. While most people enjoy their content with more passive media I find a lot more productive to read actual text.
The trues is that this blog is not even meant to be read by anyone, I simply want to document my journey and if it can provide some usefull information to anyone then it is a bonus.

The begining

In 2019 I started to have an idea, a tiny idea: Why not changing career. This tiny idea came from the fact that I always wanted to learn to code but actualy never really started. I left this tiny idea at the back of my mind for a year, then it came to me again in the moment when work got hard and I questionned my life choices. It is interesting how a small tought we had years ago can grow into something almost obscessive.
I tried to push this idea again and again saying to myself You need to focus on what you are doing now. But I could not remove this idea from my head, it even started to get clearer:Why not learning to code and make it my new career. I started reading about people starting to learn code and how you can become a develloper if you work hard enough. It's easy to find a lot of information about the subject on internet, too much to be honest.

Where to start ?

I didn't know where to start and a google search do not really answer the question. So I started looking on forums and reddit, I found sub reddit r/learnprogramming that provide a pretty list of questions on the subject. But the programing world is vast so first I needed to pick a language (C C++ c# python PHP Java Swift .NET ...
I picked html and css to start easy and because I wanted to create visual quickly as data visualisation is something I do everyday for my current job. I started on FREECODECAMP because you could jump right in the code, but I need an introduction to the subject so I bought a book "Head First HTML CSS". It was usefull to just get into it at first, but clearly the book was just scratching the paint so I got back to FREECODECAMP.
My progression on the first module was really quick as the exercice where mostly immediate repetition of the lessons.
